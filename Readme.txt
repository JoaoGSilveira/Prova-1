##---------------------------------------------Prova de GitHub---------------------------------------------##

O GitHub é uma ótima ferramenta para ter controle sobre os seus arquivos e arquivos de projeto,
tanto sozinho como em grupo/colaboração de outros amigos ou colegas de trabalho sem a necessidade
de usar dispositivos internos como HD, PEN-DRIVE, ETC...

O GitHub tambêm possui os forks, assim cada pessoa pode personalizar um software de código aberto
para sí mesmo do jeito que quiser e ainda ter os arquivos salvos nos seus repositórios.
